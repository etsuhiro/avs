cg

Character Graphic

機能

cgコマンドは画像を作成して表示、または破棄します。

パラメーター

id

画像の識別名を指定します。ハッシュ値に変換されます。
既に存在する識別名を指定すると、その画像を破棄します。

file

ファイル名を指定します。指定されたファイルをテクスチャーとして画像を作成します。

parent

親の画像の識別名を指定できます。

x

x座標を数値で指定します。画面の中央が0、左が負の値、右が正の値です。
parent属性の指定がある場合、parent画像からのオフセットになります。
省略した場合のデフォルト値は0です。

y

y座標を数値で指定します。画面の中央が0、上が負の値、下が正の値です。
parent属性の指定がある場合、parent画像からのオフセットになります。
省略した場合のデフォルト値は0です。

r

画像の赤成分を数値で指定します。
0～1.0の範囲で指定可能で、元のテクスチャーのR値と乗算されます。
parent属性の指定がある場合、parent画像のR値と乗算されます。
省略した場合のデフォルト値は1.0です。

g

画像の緑成分を数値で指定します。
0～1.0の範囲で指定可能で、元のテクスチャーのG値と乗算されます。
parent属性の指定がある場合、parent画像のG値と乗算されます。
省略した場合のデフォルト値は1.0です。

b

画像の青成分を数値で指定します。
0～1.0の範囲で指定可能で、元のテクスチャーのB値と乗算されます。
parent属性の指定がある場合、parent画像のB値と乗算されます。
省略した場合のデフォルト値は1.0です。

a

画像の透明度を数値で指定します。
0～1.0の範囲で指定可能で、元のテクスチャーのα値と乗算されます。
parent属性の指定がある場合、parent画像のα値と乗算されます。
省略した場合のデフォルト値は1.0です。

u

テクスチャーの開始位置のx座標（u値）を整数で指定します。
テクスチャーの左上が原点(0,0)になります。
省略した場合のデフォルト値は0です。

v

テクスチャーの開始位置のy座標（v値）を整数で指定します。
テクスチャーの左上が原点(0,0)になります。
省略した場合のデフォルト値は0です。

w

テクスチャーの幅のピクセル数を整数で指定します。
省略した場合のデフォルト値はテクスチャーの横サイズになります。

h

テクスチャーの高さのピクセル数を整数で指定します。
省略した場合のデフォルト値はテクスチャーの縦サイズになります。

hscale

回転が掛かる前の画像の横方向に対する拡大縮小率を数値で画像に指定します。
1.0でテクスチャーと等倍になります。
parent属性の指定がある場合、parent画像のhscale値と乗算されます。
省略した場合のデフォルト値は1.0です。

vscale

回転が掛かる前の画像の縦方向に対する拡大縮小率を数値で画像に指定します。
1.0でテクスチャーと等倍になります。
parent属性の指定がある場合、parent画像のvscale値と乗算されます。
省略した場合のデフォルト値は1.0です。

rot

画像の回転を時計回りに度数法の数値で指定します。
0～360を超えた大きな値や負の値でも指定できます。
省略した場合のデフォルト値は0です。

parentによる親子関係が設定されている場合のrotによる回転について

親がrot指定で回転すると、子も親の座標を中心に回転します。
さらに子は自分の座標を中心に自分のrot指定による回転を行います。

priority

画像の描画プライオリティーを整数で指定します。
priorityの数値が低いものから先に描画され、値が同じ場合はcgコマンドで作成された順に描画されます。
省略した場合のデフォルト値は0です。
